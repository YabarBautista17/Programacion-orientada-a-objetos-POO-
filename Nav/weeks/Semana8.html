Semana 7 - Proyecto Dashboard Visual C++

Resumen

Se desarrolló un panel gráfico usando Visual Studio 2022 con C++ CLR, que permite navegar entre distintas ventanas como Inicio, Calculadora y Fórmulas. Cada botón abre su respectiva interfaz de forma dinámica dentro del panel contenedor.

Objetivo

Integrar múltiples formularios en una sola ventana usando programación orientada a objetos en C++ y formularios CLR, orientado al uso práctico en Ingeniería Electrónica.

Ventanas Implementadas

  • Inicio: Pantalla principal del sistema.
  • Calculadora: Incluye operaciones básicas (+, −, ×, ÷) con interfaz moderna y diseño adaptable.
  • Fórmulas: Muestra fórmulas comunes en Ingeniería Electrónica (ley de Ohm, potencia, impedancia, etc).

Pasos para su desarrollo

  1. Crear un nuevo proyecto en Visual Studio 2022 usando la plantilla CLR - Windows Forms.
  2. Añadir formularios: Inicio.h, Calculadora.h y Formulas.h.
  3. Diseñar un panel lateral con botones, utilizando Dock y propiedades de estilo.
  4. Agregar los botones: Inicio, Calculadora y Formulas.
  5. Codificar los formularios con su funcionalidad respectiva (interacción y lógica).
  6. Usar una función AbrirFormulario() para cargar dinámicamente los formularios en el contenedor.

Características

- Aplicación modular y escalable
- Interfaz intuitiva y adaptable
- Incluye calculadora funcional
- Biblioteca de fórmulas básicas electrónicas

Conclusión

Este proyecto introduce el uso práctico de la programación orientada a objetos con interfaces gráficas, representando una base sólida para aplicaciones más complejas en ingeniería.
